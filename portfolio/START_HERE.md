## 🚀 QUICK START (5 MINUTES)

```bash
# 1. Open this file in your terminal/editor:
cd C:\Users\Karthik\.vscode\fastapipractice\portfolio

# 2. Read EASY_BEGINNER_GUIDE.md (first 20 minutes section)

# 3. Install FastAPI:
pip install fastapi uvicorn[standard]

# 4. Create main.py with this code:
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
async def hello():
    return {"message": "Hello World"}

# 5. Run it:
fastapi dev main.py

# 6. Open browser:
http://localhost:8000

# Done! You've created your first API! 🎉
```
